Event Management Backend API developed using Flask, PostgreSQL, SQLAlchemy, and JWT Authentication. This project is designed to manage events efficiently with secure user authentication and role-based access control.

The system allows users to register, log in securely using JWT tokens, create events, and manage event workflows. It also includes an approval system where authorized users can approve or reject events with comments.

Key Features:

* User Registration & Login
* JWT-Based Authentication
* Role-Based Authorization
* Event Creation & Management
* Event Approval Workflow
* PostgreSQL Database Integration
* RESTful API Architecture
* Database Migration Support using Flask-Migrate

Tech Stack:

* Python
* Flask
* PostgreSQL
* SQLAlchemy
* Flask-JWT-Extended
* Flask-Migrate
* Postman for API Testing

This project follows a modular backend structure with separate configurations, extensions, models, and routes, making it scalable and easier to maintain. It is suitable for learning backend development, authentication systems, database handling, and API development using Flask.
